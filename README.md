# Bulkimagen — Batch AI Image Generation

Tools, prompt templates, and examples for generating product images **in bulk** instead of one at a time — built around [Bulkimagen](https://bulkimagen.com), an AI tool for batch image generation aimed at designers and e-commerce sellers.

## The idea

Stop treating the image as the unit of work. Treat the **set** as the unit of work:

- **Lock the constant** — lighting, angle, background, style — as a reusable template
- **List the variable** — the specific product or subject
- **Generate the whole set in one pass**, so consistency is enforced by construction
- **Bulk export** with filenames that map to your catalog

## What's here

- Prompt templates for product photography and ad creatives
- Example batches and before/after results
- Notes on keeping style consistent across large sets

Built with [Bulkimagen](https://bulkimagen.com) → generate hundreds of consistent, on-brand images in minutes.
